# idTip
**idTip** is a continuation of the addon known as SpellID in WotLK.

## Features  
- Adds ID numbers to tooltips for spells, auras, items, NPCs, achievements, and quests.
- Displays the source name for aura tooltips when a valid unitID exists.
- Creates minimal tooltips when hovering over achievement frames or selecting quests in the quest log.

## Installation  
1. [Download](https://github.com/KhalGH/idTip-WotLK/releases/download/v2.01/idTip.zip) the addon
2. Extract the **idTip** folder into `World of Warcraft/Interface/AddOns/`.  
3. Restart the game and enable the addon.
